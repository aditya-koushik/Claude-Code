```

├── agastya/
│   └── AGENT.md
├── krishna/
│   └── AGENT.md
├── nakula/
│   └── AGENT.md
├── ved-vyasa/
│   └── AGENT.md
├── ... (21 more agents)
└── yudhishthira/
    └── AGENT.md
```

Each `AGENT.md` file contains:
- **Frontmatter:** id, name, type, description, codename, version, created, tools
- **Identity:** Agent name, ID, codename
- **Role Description:** What the agent does
- **Skill List:** Pre-assigned skills
- **Activation Conditions:** When to use
- **Capabilities:** What it can do
- **Frameworks:** Decision frameworks and processes
- **Integration:** How it works with other agents
- **Constraints:** Boundaries and limitations
