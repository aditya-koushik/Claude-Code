# Banned Words Extended List

## Full Banned Words and Phrases with Replacements

Every word and phrase on this list is banned from all Bit Foundry AI content. Zero tolerance. If a scan finds any of these, replace immediately.

---

## Tier 1: Core Banned Words (AI Tell-Words)

These are the words that most reliably signal AI-generated content. Finding even one of these in a piece is a quality gate failure.

| Banned Word | Replacement(s) | Example |
|------------|----------------|---------|
| delve | dig into, look at, break down, examine | "Let's dig into the data" not "Let's delve into the data" |
| dynamic | fast-moving, changing, active | "Fast-moving market" not "dynamic market" |
| unlock | open up, enable, create, find | "Open up new revenue" not "unlock new revenue" |
| revolutionize | change, transform, reshape, overhaul | "AI will change how teams work" not "AI will revolutionize how teams work" |
| harness | use, apply, put to work, deploy | "Use AI for lead scoring" not "harness AI for lead scoring" |
| comprehensive | complete, full, thorough, end-to-end | "A complete solution" not "a comprehensive solution" |
| landscape | space, market, industry, field | "The AI market" not "the AI landscape" |
| paradigm | model, framework, approach, way of thinking | "A new model" not "a new paradigm" |
| synergy | combined effect, combined power, working together | "The combined effect of sales and marketing" not "the synergy between sales and marketing" |
| leverage (verb) | use, apply, deploy, put to work | "Use your data" not "leverage your data" |
| utilize | use | Always "use." Never "utilize." No exceptions. |
| realm | area, field, space, domain | "In the AI space" not "in the realm of AI" |

## Tier 2: Marketing Buzzwords (Empty Superlatives)

These words promise everything and say nothing. Replace with specifics.

| Banned Word | Replacement(s) | Notes |
|------------|----------------|-------|
| cutting-edge | (describe the specific capability) | "Detects anomalies in 47 seconds" not "cutting-edge detection" |
| game-changer | (describe the specific result) | "Cut processing time by 82%" not "a game-changer for processing" |
| innovative | (describe what is new and why it matters) | Only use if followed by proof in the same sentence |
| transformative | (describe the transformation with numbers) | "Went from $2M to $7M revenue" not "transformative growth" |
| groundbreaking | (describe what it broke and what replaced it) | Banned unless you can name the specific ground that broke |
| disruptive | (describe what it displaced) | "Replaced 3 manual steps" not "disruptive technology" |
| world-class | (describe the specific standard) | "99.99% uptime" not "world-class reliability" |
| best-in-class | (describe the comparison) | "Faster than Datadog by 3x" not "best-in-class speed" |
| state-of-the-art | (describe the specific technology) | "Uses transformer architecture with 128K context" not "state-of-the-art AI" |
| next-generation | (describe what generation it replaces and why) | Almost always filler. Cut it. |
| robust | strong, reliable, solid | "Reliable error handling" not "robust error handling" |
| scalable | (describe the actual scale) | "Handles 4.2B API calls per day" not "scalable architecture" |
| seamless | smooth, easy, simple | "Setup takes 15 minutes" not "seamless integration" |
| holistic | complete, full, whole | "A full view of your pipeline" not "a holistic view of your pipeline" |
| empower | help, enable, let, give | "Let your team ship faster" not "empower your team" |

## Tier 3: Hedge Words (Confidence Killers)

These words soften every claim. They signal that the writer is not sure. Cut them.

| Banned Word/Phrase | Replacement | Notes |
|-------------------|-------------|-------|
| might | will, does, can | Commit or delete |
| perhaps | (delete) | State directly |
| arguably | (state directly) | If it is arguable, argue it. Do not announce that it is arguable. |
| possibly | (delete or commit) | "This will save you $50K" not "this could possibly save you $50K" |
| potentially | (delete) | "Saves $50K per year" not "potentially saves $50K per year" |
| somewhat | (delete or quantify) | "Faster" or "3x faster" not "somewhat faster" |
| relatively | (quantify instead) | "40% cheaper" not "relatively inexpensive" |
| fairly | (quantify instead) | "Takes 2 hours" not "fairly quick to set up" |
| quite | (delete or quantify) | "Strong results" or "82% improvement" not "quite good results" |
| rather | (delete) | "Important" not "rather important" |
| in some cases | (specify the case) | "For teams over 50 people" not "in some cases" |
| to a certain extent | (quantify) | "Cuts costs by 30-40%" not "cuts costs to a certain extent" |
| it seems like | (state directly) | "AI adoption is growing" not "it seems like AI adoption is growing" |
| it appears that | (state directly) | Same as above |
| tends to | does, will | "This approach works" not "this approach tends to work" |

## Tier 4: Filler Transitions (Empty Connectors)

These words take up space without adding meaning. Delete them.

| Banned Phrase | Replacement |
|--------------|-------------|
| Moreover | (delete, start next sentence) |
| Furthermore | (delete, start next sentence) |
| Additionally | (delete, start next sentence) |
| In addition | (delete, start next sentence) |
| However | But (or delete) |
| Nevertheless | But, Still |
| Consequently | So |
| Subsequently | Then |
| In conclusion | (delete, just state the conclusion) |
| To summarize | (delete, just state the summary) |
| As mentioned above | (delete, the reader remembers) |
| It is worth noting that | (delete, just state it) |
| It is important to note that | (delete, just state it) |
| It goes without saying | (if it goes without saying, do not say it) |
| Needless to say | (same as above) |
| At the end of the day | (delete or replace with "ultimately") |
| When all is said and done | (delete) |
| By and large | (delete, be specific) |
| For what it is worth | (delete, just say it) |

## Tier 5: Banned Constructions (Structural Patterns)

| Banned Pattern | Fix |
|---------------|-----|
| Em-dashes (--) | Use periods, commas, or parentheses |
| "Not only... but also" | Two separate sentences |
| "Both X and Y" (when overused) | List them as separate points |
| "In today's rapidly evolving..." | Start with a specific claim |
| "It is well known that..." | Just state the fact |
| "The fact of the matter is..." | Just state the matter |
| "When it comes to..." | Cut it. Start with the subject directly. |
| "In terms of..." | Cut it. Name the thing directly. |
| "At this point in time" | Now |
| "In the event that" | If |
| "For the purpose of" | To |
| "In order to" | To |
| "Due to the fact that" | Because |
| "On a daily basis" | Daily |
| "In the near future" | Soon |
| "A large number of" | Many |

## Quick-Scan Order

When scanning content, check in this order:

1. **Tier 1 (Core AI words)** - These are the most damaging. Find first.
2. **Tier 3 (Hedge words)** - These kill confidence. Find second.
3. **Tier 5 (Constructions)** - Em-dashes and structural patterns. Find third.
4. **Tier 4 (Filler transitions)** - Easy to spot and cut. Find fourth.
5. **Tier 2 (Buzzwords)** - Replace with specifics. Find last.

If Tier 1 or Tier 3 words are found, the piece needs a full rewrite pass, not just word swaps. The presence of these words usually means the surrounding sentences need structural fixes too.
