# Test Input 02: Edge Case — Contradictory Sources

## Task
Research the following topic. Sources will likely contradict each other. The agent must handle disagreements fairly.

## Research Query
"Is fine-tuning or prompt engineering more cost-effective for enterprise AI deployments at scale?"

## Context
This is a contentious topic in the AI community. Different experts and companies have opposing views based on their own experiences and business models. The research must present both sides.

## Requirements
- At least 4 sources
- Represent BOTH sides of the debate
- Include cost data from real deployments supporting each approach
- Note where expert opinions diverge and why
- Synthesize into a nuanced conclusion, not a one-sided argument
