# Expected Output Characteristics: Test 01

## Must Exhibit

1. **5+ distinct sources cited:** Each source named with type (report, article, case study). Mix of analyst firms, news outlets, and company reports.
2. **Market size data with attribution:** A specific dollar figure for the AI voice agent market, attributed to a named report or analyst firm (e.g., "Grand View Research estimates $X billion by 2027").
3. **Adoption rate with year:** A percentage and timeframe (e.g., "47% of enterprises have piloted AI voice agents as of 2025, per Gartner").
4. **At least 2 named case studies:** Real companies (named, not "a Fortune 500 company") with specific results (cost savings, call handling improvements).
5. **ROI data from real deployments:** Specific numbers — cost savings percentages, payback periods, or efficiency gains attributed to a named source.
6. **Structured output:** Clear sections: Key Findings, Market Data, Adoption Rates, Case Studies, ROI Data, Source List.
7. **Confidence level stated:** The research indicates how confident the findings are (strong consensus vs. limited data).
8. **Recency appropriate:** Sources from 2024-2026 for a fast-moving AI topic.

## Must NOT Exhibit

1. Unattributed claims ("studies show" without naming the study)
2. All sources from the same domain or type
3. Fabricated statistics or case studies
4. Outdated sources (pre-2023) presented as current
5. Raw search result dump without synthesis
6. Missing any of the 4 required data types (market size, adoption rate, case studies, ROI)
