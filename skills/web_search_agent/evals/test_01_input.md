# Test Input 01: Standard Research Query

## Task
Research the following topic and produce a research dossier with sourced findings.

## Research Query
"What is the current state of AI voice agent adoption in enterprise customer service? Include market size, adoption rates, key players, and ROI data."

## Context
This research will feed into a LinkedIn post about AI voice agents for Bit Foundry AI's content pipeline. The audience is enterprise CTOs.

## Requirements
- At least 5 sources
- Market size data with source attribution
- Adoption rate data with year and source
- At least 2 named enterprise case studies
- ROI or cost savings data from real deployments
