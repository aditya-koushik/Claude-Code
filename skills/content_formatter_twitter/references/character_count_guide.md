# Twitter/X Character Count Guide

## The Hard Limit

**280 characters maximum per tweet.** No exceptions. If a tweet hits 281 characters, cut a word. Do not try to squeeze it. Cut.

---

## What Counts Toward the Limit

| Element | Character Count |
|---------|----------------|
| Regular text | 1 character each (letters, numbers, spaces, punctuation) |
| URLs (any length) | 23 characters (Twitter wraps all URLs to t.co, always 23 chars) |
| @mentions | Exact character count of the handle (e.g., @bitfoundryai = 13 chars) |
| Emojis | 2 characters each (do not use them, but know the count) |
| Hashtags | Full text including the # symbol (e.g., #AI = 3 chars, #EnterpriseAI = 13 chars) |
| Line breaks | 1 character each |
| Thread numbering ("1/") | 2-3 characters including the space after |

---

## Character Budget for Threaded Tweets

For a tweet in a thread with numbering:

```
Total budget:           280 characters
Thread number ("3/ "):   -3 characters
Usable content:         277 characters
```

For the final tweet with CTA + hashtags:

```
Total budget:           280 characters
Thread number ("6/ "):   -3 characters
CTA text:              ~60-80 characters
Hashtags (5-8):        ~60-100 characters
Remaining for content:  ~97-157 characters
```

**Practical rule:** Keep the final tweet's non-hashtag content under 180 characters to leave room for 5-8 hashtags.

## Common Character Count Pitfalls

### Pitfall 1: "It looks short but it is over 280"
Spaces count. Punctuation counts. Every character counts. Do not estimate. Count.

**Fix:** Use a character counter tool. Always verify before publishing.

### Pitfall 2: URLs look short but count as 23
A link like `bit.ly/abc` looks like 11 characters. Twitter counts it as 23.

**Fix:** Budget 23 characters for any URL, regardless of its actual length.

### Pitfall 3: Mentions eat more characters than expected
Tagging @SatyaNadella is 14 characters. Tagging 3 people can eat 40+ characters.

**Fix:** Minimize mentions. If you must tag someone, count the exact characters of each handle.

### Pitfall 4: Thread numbers add up
"1/ " is 3 characters. "10/ " is 4 characters. In a 12-tweet thread, you lose 3-4 characters per tweet just on numbering.

**Fix:** Always include the numbering prefix in your character count.

### Pitfall 5: Hashtags in the final tweet run long
Eight hashtags with spaces can easily hit 90-120 characters. That leaves only 160-190 characters for your CTA and any closing text.

**Fix:** Pre-calculate hashtag character count. Write the CTA to fit the remaining space. Choose shorter hashtags if needed (#AI over #ArtificialIntelligence).

### Pitfall 6: Copy-pasting from a document adds hidden characters
Some text editors insert smart quotes, em-dashes, or zero-width spaces. These count as characters but are invisible.

**Fix:** Type directly into the composer or use a plain text editor. Replace smart quotes with straight quotes. Replace em-dashes with periods or commas.

### Pitfall 7: Forgetting line breaks count
A blank line between paragraphs within a single tweet costs 2 characters (newline + newline).

**Fix:** Budget for line breaks if you use them within a tweet. A tweet with 2 line breaks has 278 characters of usable text, not 280.

## Pre-Publish Checklist

1. Count characters for every tweet. No tweet exceeds 280.
2. URLs counted as 23 characters each, not their display length.
3. Thread numbering included in the character count.
4. Hashtags only in the final tweet. 5-8 hashtags.
5. No hashtags in tweets 1 through n-1.
6. Final tweet CTA + hashtags fits within 280 characters.
7. No em-dashes, no smart quotes, no hidden characters.
8. Each tweet is a complete thought. No sentence split across tweets.
9. Line breaks counted in the character budget.
10. Tweet 1 works as a standalone tweet if seen outside the thread.
