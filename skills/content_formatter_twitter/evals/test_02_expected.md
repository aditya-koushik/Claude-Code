# Expected Output Characteristics: Test 02 (Edge Case)

## Must Exhibit

1. **All tweets <= 280 characters:** URL counts as 23 characters (t.co wrapping), @mention counts at face value.
2. **URL preserved:** "bitfoundry.ai/voice-report-2026" appears in the thread (preferably its own tweet or combined with CTA).
3. **@mention preserved:** "@elikidesignstudio" appears exactly as written.
4. **Key stats preserved:** 340%, 12% to 47%, 67%/52%/41% use case breakdown. At least the most impactful numbers survive.
5. **Hook tweet standalone:** Does not start with "Just published our analysis..." (too bland). Reframes with a compelling stat like "The AI voice agent market grew 340% this year."
6. **3 takeaways formatted as separate tweets or clearly delineated:** Not crammed into one tweet. Each takeaway gets breathing room.
7. **Numbered thread format:** 1/, 2/, etc.
8. **Hashtags in final tweet only:** 2-3 hashtags.
9. **Thread length 6-10 tweets:** Content is dense enough to warrant 6+ tweets.

## Must NOT Exhibit

1. Any tweet over 280 characters (accounting for URL as 23 chars)
2. URL character count treated at face value instead of t.co 23-char rule
3. @mention altered or removed
4. Hook tweet that requires context from tweet 2
5. Stats presented without context (raw numbers without what they measure)
6. Hashtags in non-final tweets
