# Eval Criteria: content_formatter_twitter

## Scoring Rubric

| # | Criterion | Weight | Description |
|---|-----------|--------|-------------|
| 1 | Character Limit Compliance | 25% | Every tweet is 280 characters or fewer, including numbering prefix. Zero tolerance for exceeding 280. |
| 2 | Hook Tweet Strength | 20% | Tweet 1 works as a standalone viral tweet. Uses number, contrarian take, or bold claim. Must be compelling without the rest of the thread. |
| 3 | Thread Structure | 20% | Numbered format (1/, 2/, ..., n/). Each tweet is one complete thought. No sentence split across tweets. Logical flow from hook to conclusion. |
| 4 | Hashtag Compliance | 15% | 2-3 hashtags in the FINAL tweet only. Zero hashtags in all other tweets. |
| 5 | CTA Presence | 10% | Final tweet includes a call-to-action (follow, retweet, reply, or link). |
| 6 | Thread Length | 10% | Thread is 3-10 tweets. Not a single tweet. Not more than 25 tweets. Sweet spot is 5-8. |

**Total: 100%**

## Scoring Scale
- 0-2: Major violations (tweets over 280 chars, no structure)
- 3-4: Structure present but significant issues
- 5-6: Adequate but not optimized for engagement
- 7-8: Well structured with minor issues
- 9-10: Perfect thread formatting, every tweet standalone-quality

## Pass Threshold
**Composite score >= 8.0/10**

## Automatic Fail Conditions
- ANY tweet exceeds 280 characters = automatic fail
- Hashtags appear in non-final tweets = automatic fail
- Hook tweet (1/) is not a complete standalone thought = automatic fail
- Thread has fewer than 3 tweets = automatic fail
- A sentence is split across two tweets = automatic fail
