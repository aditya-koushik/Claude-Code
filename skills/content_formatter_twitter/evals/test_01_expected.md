# Expected Output Characteristics: Test 01

## Must Exhibit

1. **Every tweet <= 280 characters:** Count must include the numbering prefix (e.g., "1/ "). Zero exceptions.
2. **Tweet 1 is a standalone viral hook:** Something like "The consulting industry is about to have its biggest comeback in 30 years." Works on its own without context.
3. **Numbered format:** 1/, 2/, 3/, etc. Consistent prefix across all tweets.
4. **5-8 tweets total:** Content fits naturally in this range. Not over-split into 15 tweets.
5. **Each tweet = one complete thought:** No sentence fragments split across tweets. Each tweet can be read independently.
6. **Key data preserved:** "$300/hour", "$0.03", "4 minutes", "McKinsey, BCG, and Bain" must appear in the thread.
7. **Hashtags ONLY in final tweet:** 2-3 hashtags like #AI #Consulting #FutureOfWork. Zero hashtags in tweets 1 through n-1.
8. **CTA in final tweet:** "Follow for more" or "RT if you agree" or similar. Combined with hashtags.
9. **Logical flow:** Hook (attention) → Contrarian take (reframe) → Data/proof → Implication → CTA.

## Must NOT Exhibit

1. Any tweet exceeding 280 characters
2. Hashtags in any tweet except the final one
3. A sentence split mid-thought across two tweets
4. More than 3 hashtags
5. Missing tweet numbers
6. Hook tweet that requires reading tweet 2 to make sense
