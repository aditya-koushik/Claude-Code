# End of Session Command

Summarize the work session and suggest improvements.

## Instructions

1. Review all changes made in this session
2. Summarize accomplishments
3. Note any issues encountered
4. Suggest improvements to workflows/CLAUDE.md
5. Ensure clean git state

## Session Summary Format

### Accomplishments
- [What was completed]
- [Key features/fixes implemented]
- [Tests added]

### Issues Encountered
- [Any blockers or problems]
- [How they were resolved]

### Learnings
- [Patterns worth remembering]
- [Mistakes to avoid]

### Suggested Improvements
- [Updates to CLAUDE.md]
- [New slash commands needed]
- [Workflow refinements]

### Next Steps
- [What's left to do]
- [Priority items]

## After Summary

1. Commit any remaining changes
2. Update MEMORY.md if new patterns emerged
3. Ensure clean working directory
