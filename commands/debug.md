# Debug Command

Diagnose and fix errors with a systematic approach.

## Instructions

1. **Gather Context**
   - Get the error message/stack trace
   - Understand what was being attempted
   - Check recent changes

2. **Analyze**
   - Trace the control flow
   - Identify root cause
   - Check for related issues

3. **Fix**
   - Implement minimal fix
   - Add tests if applicable
   - Verify fix works

4. **Document**
   - Note the issue and fix
   - Update memory if pattern is reusable

## Debugging Approach

1. Read error message carefully
2. Find the failing line in code
3. Understand why it failed
4. Check surrounding context
5. Implement targeted fix
6. Test the fix
7. Commit with checkpoint

## Output Format

**Root Cause:** [What caused the error]

**Fix:** [What was changed]

**Prevention:** [How to avoid in future]
