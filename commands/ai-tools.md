# AI Tools Research Command

Research and report on the latest AI tools and releases.

## Instructions

1. Search for recent AI tool releases (last 30 days)
2. Focus on tools relevant to:
   - Enterprise AI
   - Developer productivity
   - AI-powered applications
   - LLMs and models
3. Provide concise summary with sources

## Sources to Check

- Product Hunt AI category
